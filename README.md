## About the Project
This is a Python-based market intelligence and reporting system built using Streamlit.  
It fetches live stock market data via APIs, stores it, and visualizes insights through an interactive dashboard.

## Tech Stack
- Python
- Streamlit
- Pandas
- Requests
- SQLite

## Features
- Live stock data fetching
- Data storage
- Interactive dashboard
- Deployed on Streamlit Cloud
